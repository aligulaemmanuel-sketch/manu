# Weather App

A Python application to fetch weather details and plot humidity forecasts using the OpenWeatherMap API.

## Features

- Get current weather details for a location
- Retrieve city IDs
- Plot 3-day humidity forecast

## Setup

1. Install dependencies: `pip install -r requirements.txt`
2. Get an API key from OpenWeatherMap and replace 'YOUR_API_KEY' in config.py
3. Run the app: `python main.py`
